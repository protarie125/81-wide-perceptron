*** 81percep.lisp
웨이트 입출력 기능은 없습니다. 테스트용 코드입니다.

1. (get-output input)
인풋에서 아웃풋을 출력합니다. (테스트용 인풋이 10개 준비되어 있습니다. *test1* ~ *test10*)
인풋은 길이 82의 리스트이어야합니다. 인풋의 0번째 값은 항상 1로 합니다(쓰레스홀드).
아웃풋은 길이 82의 리스트로 출력됩니다(아웃풋의 0번째 값은 nil입니다).

2. (learn-percep input target)
오류역전파 알고리즘을 통해 퍼셉트론을 학습합니다.
각각의 테스트용 인풋에 대한 목표 아웃풋이 준비되어 있습니다. *target1* ~ *target10*
(l-p-t input target n) 명령으로 n 번 반복학습합니다.

3. (limit-print output)
1/3 미만의 값은 0, 2/3를 초과하는 값은 1로 판정하여 아웃풋을 정리해줍니다.
실제로 값을 바꾸지는 않습니다.

4. (lpt150)
테스트용 인풋 1~10까지를 각각 150번씩 학습하고, 정리하여 출력합니다.
